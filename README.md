## Commit Message Convention

|    Type     | Description  |
|:-----------:|---|
|   `Feat`    | 새로운 기능 추가 |
|    `Fix`    | 버그 수정 |
|    `Ci`     | CI관련 설정 수정 |
|   `Docs`    | 문서 (문서 추가, 수정, 삭제) |
|   `Style`   | 스타일 (코드 형식, 세미콜론 추가: 비즈니스 로직에 변경 없는 경우) |
| `Refactor`  | 코드 리팩토링 |
|   `Test`    | 테스트 (테스트 코드 추가, 수정, 삭제: 비즈니스 로직에 변경 없는 경우) |
|   `Chore`   | 기타 변경사항 (빌드 스크립트 수정 등) |
